Notes

Block quote tag <blockquote>
This is used to add a quotation from another source.
Bold text tag <b>
Surrounding text with the bold text tag allows to darken letters to add emphasis.
Page footer tag <footer>
A footer is used usually at the bottom of a website containing copyright information, contact, authors, and many more!
Page header tag <header>
A header is a container that usually introduces the website at the very top of the page containing different headings.
Image tag <img>
The image tag is used to embed images into websites. It is also able to resize images if needed.
